# 🚀 Introduction

**TrailPaths** lets players leave a trail of blocks behind them while they walk. You can set it up to change any block they’re walking on into something else, which creates cool paths showing where people have traveled.

### 🌟 Key Features

* Customizable Trails: Define exactly which block types should be replaced and what they should turn into.
* Randomized Patterns: Use percentage-based chances to vary the appearance of your paths (e.g., a mix of Dirt and Coarse Dirt).
* Automatic Reset: Optionally configure the paths to revert to their original state after a set period, keeping your world clean.
* Toggleable: Players can easily turn their personal trail on or off whenever they like.
